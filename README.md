# Pyko - Next.js Project

Проект перенесен на Next.js 15 с TypeScript и App Router с использованием лучших практик 2026 года.

## Структура проекта

```
pyko-dev/
├── app/                    # Next.js App Router
│   ├── layout.tsx         # Root layout
│   └── page.tsx           # Главная страница
├── components/            # React компоненты
│   ├── layout/           # Layout компоненты (Header, MobileMenu)
│   └── ui/               # Переиспользуемые UI компоненты
│       ├── Button/
│       ├── Link/
│       ├── Container/
│       ├── Section/
│       ├── SectionLabel/
│       ├── SectionTitle/
│       └── ProgressBar/
├── sections/             # Секции страницы
│   ├── HeroSection/
│   ├── AboutSection/
│   └── PredictSection/
├── styles/               # Глобальные стили
│   └── globals.css
├── public/               # Статические файлы
│   └── images/          # Изображения и иконки
├── fonts/               # Шрифты
└── package.json
```

## Архитектура

### Компоненты

- **UI компоненты** (`components/ui/`) - переиспользуемые компоненты с CSS модулями
- **Layout компоненты** (`components/layout/`) - Header и MobileMenu
- **Секции** (`sections/`) - модульные секции страницы с собственными стилями

### Стили

- **Глобальные стили** (`styles/globals.css`) - шрифты, переменные, базовые стили
- **CSS модули** - каждый компонент/секция имеет свой CSS модуль для изоляции стилей

### TypeScript

- Строгая типизация всех компонентов
- Типы для props компонентов
- Полная поддержка TypeScript

## Установка и запуск

```bash
# Установить зависимости
npm install

# Запустить dev сервер
npm run dev

# Сборка для production
npm run build

# Запустить production сервер
npm start
```

## Компоненты

### UI компоненты

- `Button` - кнопка с вариантами (primary, secondary) и опцией стрелки
- `Link` - ссылка с вариантами стилей
- `Container` - контейнер для контента
- `Section` - обертка для секций с фоновым изображением
- `SectionLabel` - метка секции
- `SectionTitle` - заголовок секции с поддержкой serif шрифта
- `ProgressBar` - прогресс-бар с анимацией

### Секции

- `HeroSection` - главная секция с карточками и параллакс эффектом
- `AboutSection` - секция "О нас"
- `PredictSection` - секция с предсказательными карточками и графиками

## Изображения

Изображения должны находиться в `public/images/`:
- `public/images/temp/` - фоновые изображения секций
- `public/images/icons/` - иконки

## Шрифты

Шрифты находятся в папке `fonts/` и подключаются через `globals.css`.

## Расширение проекта

Для добавления новых страниц:
1. Создайте новую папку в `app/` (например, `app/about/page.tsx`)
2. Используйте существующие компоненты из `components/ui/` и `sections/`
3. Создайте новые секции при необходимости в `sections/`
